The boids in the flock are following 3 rules:

 * separation: try not to hit other boids within a radius
 * alignment: move to the average heading of others within a radius
 * cohesion: move to the average position of others within a radius

The main resource by Craig Reynolds: https://www.red3d.com/cwr/boids/
Detailed explanation and psuedo code: http://www.kfish.org/boids/pseudocode.html
